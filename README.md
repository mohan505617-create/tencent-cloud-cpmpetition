## 🎓 项目简介

EduAI Hub 是专为香港城市大学打造的综合教育平台，整合了知识管理、研究分析和教学工具，为学生、研究者和教师提供一站式的智能化学术支持。

### 🌟 核心理念
- **数字遗产传承**: 继承并发展学校的教育传统和学术精神
- **智能化支持**: AI驱动的学习和教学辅助功能
- **一站式服务**: 统一平台整合所有教育资源
- **现代化体验**: 美观直观的用户界面设计

## 🚀 快速开始

### 环境要求
- Node.js 16.0 或更高版本
- npm 或 yarn 包管理器
- 现代浏览器 (Chrome, Firefox, Safari, Edge)

### 安装和运行

1. **克隆项目**
```bash
git clone https://github.com/your-username/eduai-hub.git
cd eduai-hub
```

2. **安装依赖**
```bash
npm install
```

3. **启动开发服务器**
```bash
npm run dev
```

4. **启动后端API服务器**
```bash
cd server
npm install
npm start
```

5. **访问应用**
- 前端应用: http://localhost:3001
- 后端API: http://localhost:8080
